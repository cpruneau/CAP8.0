enum CAP::GlauberEventFilter::GlauberEventFilterType

Filter types kAny : all Glauber accepted. kImpact: filtering based on impact parameter. kNParts: filtering based on number of participants. kNColls: filtering based on number of binary collisions. kNSpecs: filtering based on number of spectators..
