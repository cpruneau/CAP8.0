50/* Coulomb wave functions F_{lam_F}(eta,x), G_{lam_G}(eta,x)
51 * and their derivatives; lam_G := lam_F - k_lam_G
52 *
53 * lam_F, lam_G > -0.5
54 * x > 0.0
55 *
56 * Conventions of Abramowitz+Stegun.
57 *
58 * Because there can be a large dynamic range of values,
59 * overflows are handled gracefully. If an overflow occurs,
60 * GSL_EOVRFLW is signalled and exponent(s) are returned
61 * through exp_F, exp_G. These are such that
62 *
63 * F_L(eta,x) = fc[k_L] * exp(exp_F)
64 * G_L(eta,x) = gc[k_L] * exp(exp_G)
65 * F_L'(eta,x) = fcp[k_L] * exp(exp_F)
66 * G_L'(eta,x) = gcp[k_L] * exp(exp_G)
67 */
