\doxysubsection{Detailed Description}
Class encapsulates the plotting properties or attributes of a graph or histograms. This class is designed to work with other \doxylink{namespace_c_a_p}{CAP} graphics classes to set the plotting attributes of graphs (TGraph), histograms (TH1\texorpdfstring{$\ast$}{*}), and data graphs (\doxylink{class_c_a_p_1_1_data_graph}{Data\+Graph}) displaying data values, statistical error bars, and systematic error bars.

Lines are used to draw connections between points as well as (statistical) error bars. Markers are used to represent the data point position/\+value Systs are used to represent the systematic error bars or bounds.
