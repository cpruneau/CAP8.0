\doxysubsection{Detailed Description}
This class defines the notion of particle type used in \doxylink{namespace_c_a_p}{CAP}. Virtually all elementary particle properties are represented and stored. \doxylink{class_c_a_p_1_1_property}{Property} values are stored and read from a particle property file. \doxylink{class_c_a_p_1_1_property}{Property} values are based on the particle data book maintained by the particle data group (PDG).
